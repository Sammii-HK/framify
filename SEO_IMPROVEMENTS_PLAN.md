# SEO Improvement Plan for Lunary

## Priority 1: Content Depth & Individual Pages

### 1.1 Create Individual Tarot Card Pages
**Why:** Better SEO than one long page, enables specific keyword targeting
- Create `/grimoire/tarot/[card-name]` pages
- Each card gets its own URL, title, description
- Add breadcrumbs linking back to grimoire
- Enable rich snippets for individual cards

**Example URLs:**
- `/grimoire/tarot/the-fool`
- `/grimoire/tarot/ace-of-wands`
- `/grimoire/tarot/the-fool-reversed`

### 1.2 Create Individual Zodiac Sign Pages
**Why:** High search volume for individual signs
- Create `/grimoire/zodiac/[sign-name]` pages
- Detailed content: traits, compatibility, dates, ruling planet
- Add compatibility charts
- Add daily/monthly horoscope sections

**Example URLs:**
- `/grimoire/zodiac/aries`
- `/grimoire/zodiac/taurus`

### 1.3 Expand Grimoire Main Page
**Why:** More content = better rankings
- Add detailed descriptions for each section
- Add "How to Use This Guide" section
- Add related articles/guides
- Add FAQ section with FAQPage schema

## Priority 2: Structured Data Enhancements

### 2.1 Add FAQPage Schema
**Why:** Enables FAQ rich snippets in search results
- Add common astrology/tarot FAQs
- Use FAQPage structured data
- Place on grimoire and individual pages

### 2.2 Add Article Schema for Guides
**Why:** Better visibility for educational content
- Markup guide articles as Article schema
- Include author, publish date, category

### 2.3 Add Breadcrumb Navigation
**Why:** Better UX and enables breadcrumb rich snippets
- Add breadcrumbs to all pages
- Implement BreadcrumbList schema
- Visual breadcrumb component

### 2.4 Add HowTo Schema
**Why:** Rich snippets for "how to" queries
- "How to Read Tarot Cards"
- "How to Calculate Birth Chart"
- "How to Read Reversed Cards"

## Priority 5: Technical SEO Improvements

### 5.1 Image Optimization
**Why:** Better page speed, image search traffic
- Add alt text to all images
- Optimize image sizes
- Use WebP format
- Add image structured data

### 5.2 URL Structure
**Why:** Clean URLs rank better
- Use hyphens, not underscores
- Keep URLs short and descriptive
- Include keywords naturally

### 5.3 Meta Descriptions
**Why:** Better click-through rates
- Write compelling, unique descriptions
- Include call-to-action
- Keep under 160 characters
- Include primary keyword

### 5.4 Title Tag Optimization
**Why:** Critical for rankings
- Include primary keyword near front
- Keep under 60 characters
- Make each title unique
- Include brand name
